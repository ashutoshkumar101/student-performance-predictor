# Reflection

> This document contains my personal learning notes and reflections after completing this project. It summarizes what I learned, the mistakes I made, and how I would improve the project in the future.

## 1. What did I learn from this project?
- I learnt how to understand and visualise the data well.
- I learnt about the important steps in preprocessing.
- I learnt when to use One-Hot Encoding and when to use Label Encoding.
- I leant how to choose models, evaluate and compare them for a definite problem.
- I learnt about the importance of a good quality data and how a poor quality dataset can hinder our prediction.
- I learnt about the baseline models.
- I learnt the whole ML Workflow.
- I learnt how to systematically diagnose problems when they occur.
- I learnt that simpler models like Linear Regression can outperform Complex Models like Random Forest Regressor.
- I learnt that understanding the reasoning behind every preprocessing step and model choice is more important than simply knowing how to implement them.

## 2. What mistakes did I make?
- At various steps, I was using the conventional things without knowing the exact reason behind them such as One-Hot-Encoding vs Label Encoder.
- I had once preassumed that the more-flexible model like Random Forest Regression would surely outperform simpler models like Linear Regressor. 

## 3. What surprised me the most?
- How badly a poor quality dataset can influence our predictions.
- Sometimes, simpler models can outperform the complex models like in the case of Linear Regression vs Random Forest Regressor.

## 4. If I had one more week, what would I improve?
- I would have worked on improving the dataset features by introducing other important academic features like study hours, previous academic performances, sleep hours etc.
